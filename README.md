# Приложение для отдела кадров
**Описание работы**

Приложение представляет собой систему, выполненную на C# с хранением данных в **JSON-файлах**.

Система включает регистрацию, авторизацию пользователей и работу по
ролям: **кадровик, начальник отдела, бухгалтер, сотрудник**.

## 1. Регистрация

Пользователь вводит логин, пароль и роль. После регистрации создаётся
запись данных, а `EmployeeId` остаётся `null` (работник не привязан к общей базе), пока кадровик не
привяжет сотрудника.

## 2. Вход в систему

Форма входа включает в себя: список пользователей, прошедших регистрацию; поля введения логина и пароля; кнопку входа.

После входа отображается интерфейс в зависимости от роли.

## 3. Роли и их возможности

### Кадровик
-   управление сотрудниками
-   увольнения
-   формирование отчётности
-   связь пользователя и записанного сотрудника
-   просмотр отпусков всех сотрудников

### Начальник отдела
-   просмотр сотрудников своего отдела
-   утверждение отпусков

### Бухгалтер
-   информация по отпускам
-   информация по больничным

### Сотрудник
-   просмотр личных данных
-   подача заявлений на отпуск
-   просмотр истории отпусков

Если сотрудник не привязан - система сообщает об этом.

## 4. Выход из системы

Кнопка "Выйти" скрывает `MainForm` и снова показывает форму логина.
